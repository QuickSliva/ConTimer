# CoN Timer Assistant 🎖️
A lightweight Chrome Extension designed for **Conflict of Nations** players to track unit movements, research, and city captures with integrated Discord notifications.

## ✨ Features
- Set timers down to the second.
- Discord Integration: Get pinged via Webhooks when your event is complete.
- Role Pings: Choose between pinging yourself or a specific Coalition Role.
- Privacy First: All data is stored locally in your browser.

## 🚀 Installation
1. Download this repository as a ZIP and extract it.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable Developer Mode (top right).
4. Click Load unpacked and select the folder containing the files.

## ⚠️ Security Note
Your Discord Webhook URL is a sensitive key. Anyone with this URL can send messages to your server. Never share your webhook URL or upload your configured `chrome.storage` to a public place.
