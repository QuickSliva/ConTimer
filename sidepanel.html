Discord Webhook Setup
Open your server settings → Integrations → Webhooks. Create a new webhook and copy the URL.
Getting IDs
Enable 'Developer Mode' in User Settings → Advanced. Right-click a User or Role → 'Copy ID'.
Security Protocol
A webhook URL is a direct key to your server. Do not share this link.
If you suspect your webhook has been compromised, delete it immediately.
Privacy Statement:
No data is collected or sent to external servers, except for the notification sent to your specified Discord Webhook.
